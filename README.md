# 📊 Mutual Fund Analytics Project

An end-to-end **Mutual Fund Analytics Project** built using **Python, Pandas, NumPy, SQLite, Matplotlib, SciPy, and Jupyter Notebook**.

This project demonstrates a complete data analytics workflow—from data ingestion and cleaning to exploratory data analysis (EDA), financial performance analytics, and benchmark comparison.

---

## 🚀 Project Highlights

- 📥 Data Ingestion & Validation
- 🧹 Data Cleaning & Preprocessing
- 🗄️ SQLite Database Integration
- 📊 Exploratory Data Analysis (EDA)
- 📈 Performance Analytics
- 📉 Risk Metrics Calculation
- 🏆 Fund Ranking System
- 📊 Benchmark Comparison

## 📊 Project Workflow

### 1️⃣ Data Ingestion
- Loaded multiple mutual fund datasets
- Imported NAV history
- Imported benchmark indices
- Imported investor transactions

### 2️⃣ Data Cleaning
- Missing value treatment
- Duplicate removal
- Data validation
- Standardized formats

### 3️⃣ Database
- SQLite database creation
- Clean tables stored for analysis

### 4️⃣ Exploratory Data Analysis
- NAV Trend Analysis
- Fund Category Analysis
- Return Distribution
- Expense Ratio Analysis
- Risk Analysis
- Correlation Analysis

### 5️⃣ Performance Analytics
- Daily Returns
- CAGR (1-Year & 3-Year)
- Sharpe Ratio
- Sortino Ratio
- Alpha & Beta
- Maximum Drawdown
- Tracking Error
- Fund Scorecard (0–100)

### 6️⃣ Benchmark Comparison
- NIFTY50 Comparison
- NIFTY100 Comparison
- Top 5 Fund Performance
- Performance Visualization

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/kundan645/MutualFundProject.git
```

Move into the project

```bash
cd MutualFundProject
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
